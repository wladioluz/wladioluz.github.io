Definition List of HTML, CSS, FTP and SSH

HTML
HyperText Markup Language - the standard markup language for documents designed to be displayed in a web browser.¹
CSS
Cascading Style Sheets - style sheet language used for describing the presentation of a document written in a markup language such as HTML.²
FTP
File Transfer Protocol - standard communication protocol used for the transfer of computer files from a server to a client on a computer network.³
SSH
Secure Shell - cryptographic network protocol for operating network services securely over an unsecured network.⁴

¹-description from Wikipedia.
²-description from "CSS developer guide". Mozilla Developer Network. Archived from the original on 2015-09-25. Retrieved 2015-09-24".
³-description from "Forouzan, B.A. (2000). TCP/IP: Protocol Suite (1st ed.). New Delhi, India: Tata McGraw-Hill Publishing Company Limited".
⁴-description from "T. Ylonen; C. Lonvick (January 2006). The Secure Shell (SSH) Protocol Architecture. Network Working Group of the IETF".
